Параметры-переменные функций в Python

Пусть у нас есть следующая функция:

def func(num): return num ** 2

Вызовем функцию, параметром передав в нее число 2:

print(func(2))

Однако не обязательно передавать именно число - можно для этого использовать и переменную, содержащую нужное нам значение:

tst = 3 print(func(tst)) # выведет 9

Даны три переменные с числами:

tst1 = 2 tst2 = 4 tst3 = 6

Сделайте функцию, которая в свои параметры будет принимать три числа и находить их сумму. Выведите на экран сумму заданных выше переменных.

Даны функция func и переменная tst:

def func(lst): sum = 0 for el in lst: sum += el return sum tst = [1, 3, 6]
