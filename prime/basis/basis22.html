Сложение строк в Python

Для объединения (конкатенации) строк используется оператор +:

test = 'abc' + 'def' # складываем две строки print(test) # выведет 'abcdef'

Строки также могут быть сохранены в переменных:

test1 = 'abc' test2 = 'def' test3 = test1 + test2 # складываем две строки print(test3) # выведет 'abcdef'

Также можно объединять переменные и строки:

test1 = 'abc' test2 = 'def' test3 = test1 + '!!!' + test2 print(test3)

Даны две строки:

txt1 = 'abc' txt2 = 'def'

Даны следующие строки:
