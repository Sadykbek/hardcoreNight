Комментарии в Python

В Python допускаются комментарии, которые могут быть однострочными или многострочными.

Однострочные комментарии начинаются с символа хештега #:

# test

Чтобы создать многострочный комментарий в Python, необходимо включить его в тройные кавычки:

""" test1 test2 test3 """

Закомментируйте следующую строку:

test = 'abc'

Пусть у вас есть следующие строки:

test1 = 'abc' test2 = '123' test3 = 123

Заключите их в многострочный комментарий.
