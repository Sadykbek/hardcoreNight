Математические операции с числами в Python

В Python можно выполнять все основные математические операции. Давайте рассмотрим ниже каждую из них.

Сложение:

test = 1 + 2 print(test) # выведет 3

Вычитание:

test = 2 - 1 print(test) # выведет 1

Умножение:

test = 3 * 2 print(test) # выведет 6

Деление:

test = 4 / 2 print(test) # выведет 2

Деление нацело:

test = 10 // 3 print(test) # выведет 3
