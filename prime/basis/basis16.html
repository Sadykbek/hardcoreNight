Числа с плавающей точкой в Python

В Python также присутствуют числа с плавающей точкой. Они представляют собой числа, в которых целая и дробная части разделяются точкой:

num = 0.5 print(num) # выведет 0.5

С такими числами можно выполнять все математические операции. Давайте, например, сложим два числа с плавающей точкой:

res = 0.5 + 0.5 # выведет 1.0 print(res)

В переменную num1 запишите число 1.5, а в переменную num2 - число 0.75. Найдите и выведите в консоль сумму значений этих переменных.

Дан следующие числа:

num1 = 3.4 num2 = 8.27 num3 = 1.5

Отнимите от второго числа первое, а затем умножьте результат на третье число.
