Отрицательные числа в Python

Числа могут быть отрицательными. Для этого перед числом необходимо поставить знак минус:

num = -1 print(num) # выведет -1

Знак минус можно писать как к числам, так и к переменным:

num1 = 1 num2 = -num1 # записали в num2 содержимое num1 с обратным знаком print(num2) # выведет -1

Или вот так:

num = 1 print(-num) # выведет -1
