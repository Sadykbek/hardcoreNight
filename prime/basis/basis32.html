Преобразование в целые числа на Python

Для преобразования строки в число нужно использовать функцию int. Давайте сделаем из двух строковых переменных числовые и получим их сумму:

txt1 = '1' txt2 = '2' res = int(txt1) + int(txt2) print(res) # выведет число 3

Можно воспользоваться командой преобразования прямо в вызове функции:

print(int('123')) # выведет число 123

txt = '123'
