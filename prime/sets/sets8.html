Удаление элемента из множества в Python

Чтобы удалить элемент из множества, можно применить метод remove. В его параметр передаем нужный элемент.

Пусть у нас есть множество:

st = {'a', 'b', 'c'}

Давайте удалим из него элемент 'a':

st.remove('a') print(st) # выведет {'b', 'c'}

Если элемента, который нужно удалить, нет в множестве, то вернется ошибка:

st.remove('d') print(st) # выведет ошибку

Дано множество:

st = {1, 2, 3, 4, 5}

Удалите из него элемент со значением 3.

Дан следующий код:

st = {'12', 1, '34', 2, '56'} st.remove('1') print(st)

Скажите, что выведется в консоль.

Дан следующий код:

st = {1, 7, '2', 14, 5, 2} st.remove(2) print(st)

Скажите, что выведется в консоль.
