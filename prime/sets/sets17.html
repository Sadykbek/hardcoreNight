Разность множеств в Python

Чтобы найти элементы, по которым первое множество отличается от второго, нужно использовать метод difference. В его параметре указываем множество, с которым хотим сравнить исходное.

Пусть у нас есть два множества:

st1 = {'a', 'b', 'c', 'e'} st2 = {'b', 'w', 'c', 'a'}

Давайте выведем элементы, по которым первое множество отличается от второго:

res = st1.difference(st2) print(res) # выведет {'e', 'd'}

А теперь найдем элементы, по которым второе множество отличается от первого:

res = st2.difference(st1) print(res) # выведет {'w', 'c'}

В более короткой форме данный метод можно переписать вот так:

res = st2 - st1 print(res) # выведет {'w', 'c'}
