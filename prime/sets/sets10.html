Извлечение элемента из множества в Python

Чтобы извлечь случайный элемент из множества, нужно применить метод pop. В его параметр ничего не передаем.

Пусть у нас есть множество:

st = {'a', 'b', 'c'}

Давайте извлечем из него случайный элемент:

print(st.pop()) # выведет 'b'

Элемент после извлечения удаляется из множества. Давайте обратимся к исходному множеству:

print(st) # выведет {'b', 'c'}

Дано множество:

st = {1, 2, 3, 4, 5}
