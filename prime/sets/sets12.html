Наличие элемента в множестве в Python

Чтобы проверить наличие элемента в множестве, следует применить оператор in. Слева от оператора указывается нужный элемент, а справа - в каком множестве его искать. Если элемент находится в множестве, то возвращается булевое значение True, в противном случае - False.

Пусть у нас есть множество:

st = {'a', 'b', 'c'}

Давайте проверим, есть ли в нем элемент 'a':

res = 'a' in st print(res) # выведет True

А теперь попробуем найти в множестве элемент 'e':

res = 'e' in st print(res) # выведет False

Также можно проверить наличие какого-либо элемента сразу в нескольких множествах. Для это следует применить еще и оператор объединения &:

st1 = {1, 2, 3, 4} st2 = {3, 4, 5, 6} print(3 in st1 & st2) # выведет True

Чтобы пойти от обратного и узнать нет ли элемента в множестве, можно применить конструкцию not in:

st = {'1', '2', '3'} res = '4' not in st print(res) # выведет True

st = {1, 2, 3, 4, 5}
