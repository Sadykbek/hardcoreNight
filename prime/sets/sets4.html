Длина множества в Python

Найти длину множества можно с помощью функции len:

st = {'a', 'b', 'c', 'd'} print(len(st)) # выведет 4

Дано множество:

st = {'x', 1, 'y', 2, 'z', 3, 'w'}

Узнайте количество элементов в нем.

Дано множество:

st = {1, 2, 3, 4, 2, 1}

Узнайте количество элементов в нем.
