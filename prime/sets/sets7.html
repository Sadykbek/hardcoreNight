Объединение множеств в Python

Метод update позволяет не только добавлять элементы, но и объединять несколько множеств.

Пусть у нас есть два множества:

st1 = {'a', 'b', 'c', 'd'} st2 = {1, 2, 3, 4}

Давайте добавим к первому множеству второе:

st1.update(st2)

А затем выведем первое множество в консоль:

print(st1) # выведет {'c', 1, 2, 3, 4, 'b', 'd', 'a'}

В метод update можно передать и несколько множеств. Давайте создадим третье множество:

st3 = {'x', 'y', 'z'}

Теперь давайте добавим к первому множеству второе и третье:

st1.update(st2, st3)

Выведем в консоль первое множество:

print(st1) # выведет {1, 2, 3, 4, 'd', 'y', 'x', 'c', 'a', 'b', 'z'}

Объединение множеств можно записать в более сокращенном виде с помощью оператора |. Давайте перепишем предыдущий пример в краткой форме:

st1 = st2 | st3

Даны два множества:

st1 = {'a', 'b', 'c', 'd', 'e'} st2 = {'d', 'e', 'f', 'g', 'h'}

Объедините эти множества в одно.

Даны три множества:

st1 = {'2', '4', '6'} st2 = {7, 8, 9} st3 = {'1', '3', '4'}

Объедините эти множества в одно.

Даны множества:

st1 = {1, 2, 3} st2 = {'a', 'b', 'c'} st3 = {4, 5, 6} st4 = {'d', 'e', 'f'}

Объедините в краткой форме сначала первое и третье, затем второе и четвертое множества. Выведите полученный результат в консоль.
