Получение времени в формате epoch в Python

При работе с датой в Python существует специальный формат epoch, который отображает количество миллисекунд, прошедшее с 1-го января 1970 года по текущий (или заданный) момент времени. Данный формат очень удобен для передачи и хранения данных о времени.

Для получения epoch нужно сначала импортировать модуль time. А затем применить к нему специальный метод time. Давайте получим текущий момент времени в этом формате:

import time res = time.time() print(res) # выведет количество миллисекунд по текущий момент времени
