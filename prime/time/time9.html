Получение даты из формата epoch в Python

Чтобы из формата epoch вывести дату, нужно применить метод ctime. В его параметр передается epoch.

Давайте получим дату текущего момент времени, преобразовав epoch:

dt = time.time() res = time.ctime(dt) print(res) # выведет текущую дату

Выведите из формата epoch текущую дату.
