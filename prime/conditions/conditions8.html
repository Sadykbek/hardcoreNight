Логическое НЕТ в Python

Логическое НЕТ применяется для обозначения отрицания в условии. Записывается оно с помощью конструкции not.

Пусть у нас есть переменная:

tst = 2

Давайте зададим условие - больше ли переменная 0 и при этом не меньше 1:

if tst > 0 and not tst < 1: print('+++') # сработает это else: print('---')

Дан следующий код:

tst = 15 if tst > 20 and not tst < 10: print('+++') else: print('---')

Скажите, что выведется в консоль.

Дан следующий код:

tst1 = -8 tst2 = 10 if tst1 > -10 and not tst2 < 10: print('+++') else: print('---')

Скажите, что выведется в консоль.
