Логическое ИЛИ в Python

Логическое ИЛИ требует выполнения хотя бы одного условия. Записать такое условие можно с помощью оператора or.

Пусть у нас есть две переменные:

tst1 = 10 tst2 = -5

Давайте пропишем условие - если переменная tst1 больше 0 или переменная tst2 больше 0, то в консоль выведется '+++':

if tst1 > 0 or tst2 > 0: print('+++') # сработает это else: print('---')

Даны переменные:

tst1 = -1 tst2 = 4

Проверьте, что переменная tst1 равна или меньше 1, а переменная tst2 больше или равна 3.

Не запуская код определите, что выведется в консоль:

tst1 = -10; tst2 = -10; if tst1 >= 0 or tst2 >= 0: print('+++'); else: print('---'); }

Не запуская код определите, что выведется в консоль:

tst1 = 0; tst2 = 0; if tst1 >= 0 or tst2 >= 0: print('+++'); else: print('---');

Не запуская код определите, что выведется в консоль:

tst1 = 0; tst2 = 5; if tst1 >= 0 or tst2 >= 0: print('+++'); else: print('---');

Не запуская код определите, что выведется в консоль:

tst1 = 5; tst2 = 5; if tst1 >= 0 or tst2 >= 0: print('+++'); else: print('---');

Не запуская код определите, что выведется в консоль:

tst1 = -5; tst2 = 15; if tst1 >= 0 or tst1 >= 0: print('+++'); else: print('---');

Не запуская код определите, что выведется в консоль:

tst = 1; if tst == 0 or tst == 1: print('+++'); else: print('---');

Не запуская код определите, что выведется в консоль:

tst = 2; if tst == 0 or tst == 1: print('+++'); else: print('---');

Не запуская код определите, что выведется в консоль:

tst = 2; if tst == 0 or tst == 1 or tst == 2: print('+++'); else: print('---');
