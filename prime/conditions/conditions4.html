Проверка на равенство в Python

Для проверки двух значений на равенство используется оператор ==.

Пусть есть переменная:

tst = 3

Давайте узнаем равна ли она 0:

if tst == 0: print('+++') else: print('---') # сработает это

Теперь поменяем значение переменной так, чтобы условие выполнялось:

tst = 0 if tst == 0: print('+++') # сработает это else: print('---')

Дана переменная:

tst = 25

Проверьте, что она равна 10.

Дан следующий код:

tst = '123' if tst == 123: print('+++') else: print('---')

Скажите, что выведется в консоль.

Дан следующий код:

tst = ['a', 'b, 'c', 'd'] if len(tst) == 5: print('+++') else: print('---')

Скажите, что выведется в консоль.
