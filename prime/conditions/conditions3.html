Операторы больше и меньше в Python

Для сравнения двух значений между собой используются операторы больше >, больше или равно >=, меньше <, меньше или равно <=.

Пусть у нас есть переменная tst:

tst = 3

Давайте узнаем больше ли нуля значение этой переменной:

if tst > 0: print('+++') # сработает это else: print('---')

Теперь давайте поменяем значение переменной на отрицательное:

tst = -3 if tst > 0: print('+++') else: print('---') # сработает это

Пусть теперь значение переменной равно 0. В этом случае выполнится блок else:

tst = 0 if tst > 0: print('+++') else: print('---') # сработает это

Давайте поменяем условие на больше или равно:

if tst >= 0: print('+++') # сработает это else: print('---')

Если поменять условие на меньше или равно, то также сработает первый блок:

if tst <= 0: print('+++') # сработает это else: print('---')

Дана переменная:

tst = 5

Проверьте, что она больше 10.

Проверьте, что переменная tst меньше 10.

Проверьте, что переменная tst больше или равна 5.

Проверьте, что переменная tst меньше или равна -5.

Дана переменная:

tst = 5

Проверьте, что она больше 10.

if age >= 18: print('You have access to the site') else: print('Нou do not have access to the site')
