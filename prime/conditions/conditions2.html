Отступы в конструкции if-else в Python

В Python отступы определяют, какие блоки конструкции if-else объединены в пару. Поэтому строки под каждым блоком должны находится на расстоянии одного отступа. Он равен четырем пробелам или одной табуляции.

Давайте рассмотрим общую схему условия if-else. В правильном варианте код должен находится на расстоянии одного отступа от своего блока с условием:

if операция сравнения: ''' код # это сработает ''' else: ''' код # это сработает '''

Если же убрать отступ, то Python вернет ошибку:

if операция сравнения: ''' код # выведет ошибку ''' else: ''' код # этот код не сработает, потому что выше вывелась ошибка '''
