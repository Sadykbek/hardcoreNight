Проверка на неравенство в Python

Существует также оператор !=, который проверяет на неравенство.

Пусть есть переменная:

tst = 5

Давайте проверим, что она не равна 0:

if tst != 0: print('+++') # сработает это, так как переменная НЕ равна 0 else: print('---')

Теперь поменяем значение переменной:

tst = 0 if tst != 0: print('+++') else: print('---') # сработает это, так как переменная равна 0

Дана переменная:

tst = -4

Проверьте, что она не равна 4.

Дан следующий код:

tst = 123 if tst != 123: print('+++') else: print('---')

Скажите, что выведется в консоль.

Дан следующий код:

tst = ['a', 'b', 'c'] if tst[1] != 'c': print('+++') else: print('---')

Скажите, что выведется в консоль.

Дан следующий код:

tst1 = [3, 4, 5, 6] tst2 = list('3456') if tst1 != tst2: print('+++') else: print('---')

Скажите, что выведется в консоль.
