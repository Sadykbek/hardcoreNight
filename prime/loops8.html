Значения словаря через for в Python

Для получения значений словаря, можно обратиться к ним по ключу.

Пример

Пусть у нас есть словарь dct:

dct = { 'a': 1, 'b': 2, 'c': 3 }

Давайте выведем все его значения:

for key in dct: print(dct[key])

Результат выполнения кода:

1 2 3

Пример

Вывести значения словаря можно и с помощью метода values. Он возвращает специальный объект, который перебирается циклом:

for el in dct.values(): print(el)

Результат выполнения кода:

1 2 3
