Перебор многомерного списка словарей в Python

Также с помощью цикла можно перебирать многомерный список словарей. Пусть у нас есть следующий список:

lst = [ { 'a': 1, 'b': 2, }, { 'c': 3, 'd': 4, }, { 'e': 5, 'f': 6, }, ]

Давайте выведем все ключи словарей на экран. Для этого прогоним наш список через цикл два раза:

for sub in lst: for key in sub: print(key)

Дан список словарей:

lst = [ { 'a': 1, 'b': 2, 'c': 3 }, { 'a': 4, 'b': 5, 'c': 6 }, { 'a': 7, 'b': 8, 'c': 9, }, ]

С помощью цикла найдите сумму всех чисел.

Измените код предыдущей задачи так, чтобы вывести из списка пары ключ-значение словаря:

'a' 1 'b' 2 'c' 3 'a' 4 'b' 5 'c' 6 'a' 7 'b' 8 'c' 9
