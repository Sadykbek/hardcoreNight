Перебор многомерных списков в Python

Давайте теперь научимся перебирать многомерные списки с помощью циклов. Пусть дан следующий список:

lst = [ ['a', 'b', 'c'], ['d', 'e', 'f'], ['g', 'h', 'i'] ]

Поскольку данный список двухмерный, для его перебора нужно два вложенных цикла for:

for sub in lst: for el in sub: print(el)

Дан двухмерный список:

lst = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]

С помощью цикла выведите все элементы списка в консоль.

Дан двухмерный список:

lst = [ [2, 4, 6], [3, 5, 7], [9, 12, 15] ]

С помощью цикла найдите сумму элементов этого списка.

Дан двухмерный список:

lst = [ ['a', 'b', 'c'], ['d', 'e', 'f'], ['g', 'h', 'i'] ]

С помощью цикла слейте все элементы списка в строку.
