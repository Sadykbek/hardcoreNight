Перебор трехмерного списка в Python

Для перебора трехмерного списка нужно использовать три вложенных цикла. Пусть для примера у нас есть вот такой список:

lst = [ [ ['a', 'b'], ['c', 'd'], ], [ ['e', 'f'], ['g', 'h'], ] ]

Давайте выведем все его элементы на экран:

for sub in lst: for sub_el in sub: for el in sub_el: print(el)

Дан трехмерный список:

lst = [ [ ['q', 'w', 'e'], ['r', 't', 'y'], ['u', 'i', 'o'], ], [ ['p', 'a', 's'], ['d', 'f', 'g'], ['h', 'j', 'k'], ], [ ['l', 'z', 'x'], ['c', 'v', 'b'], ['n', 'm', 'q'], ], ]

С помощью цикла выведите все элементы списка в консоль.

Дан трехмерный список:

lst = [ [ [1, 3], [5, 7], ], [ [2, 4], [6, 8], ], ]
