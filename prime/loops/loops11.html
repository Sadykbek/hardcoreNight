Генерация чисел через for в Python

Для последовательного вывода чисел из заданного диапазона следует использовать функцию range в цикле for.

Давайте выведем числа от 1 до 9. Для этого передадим в первый параметр функции range единицу. Во втором параметре укажем 10, поскольку функция выбирает числа, не включая значение второго параметра:

for num in range(1, 10): print(num) # выведет 1, 2... 9
