Пара ключ-значение словаря через for в Python

С помощью цикла for можно вывести также пару ключ-значение словаря.

Пример

Пусть у нас есть словарь dct:

dct = { 'a': 1, 'b': 2, 'c': 3 }

Давайте выведем все его элементы. Для этого в функции print укажем ключ и значение:

for key in dct: print(key, dct[key])

Результат выполнения кода:

'a 1' 'b 2' 'c 3'

Пример

Получить ключи словаря можно и с помощью метода items. Метод возвращает специальный итерируемый объект, который в можно перебрать циклом:

for el in dct.items(): print(el)

После выполнения кода выведутся кортежи, состоящие из пар ключ-значение:

('a', 1) ('b', 2) ('c', 3)

Практические задачи

Дан словарь:

dct = { 'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5 }

Выведите в консоль ключи и значения элементов.

Создайте словарь, содержащий порядковый номер месяца и его имя. Выведите все пары ключ-значение этого словаря.
