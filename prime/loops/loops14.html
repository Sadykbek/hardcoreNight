Цикл while в Python

Особенностью цикла while является то, что он будет выполняться до тех пор, пока верно выражение, переданное ему.

Его синтаксис выглядит так:

while пока выражение истинно: код, который выполняет пока условие не стало ложным

В условии цикла while необходимо использовать динамически меняющиеся значения, например, счетчик, иначе может происходить бесконечный вывод значений в консоль. Давайте зададим выполнение цикла, пока счетчик будет меньше 5:

i = 0 while i < 5: i += 1 print(i)

Результат выполнения кода:

1 2 3 4 5

С помощью цикла while выведите в консоль числа от 1 до 10.
