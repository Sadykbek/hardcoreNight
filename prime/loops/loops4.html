Инструкция break в Python

Выполнение цикла можно прервать с помощью инструкции break.

Пусть у нас есть список lst:

lst = [1, 2, 3, 4, 5]

Давайте выведем из него все элементы до числа 3, на котором прервется выполнение цикла. Для этого под блоком с условием прописываем инструкцию break:

for el in lst: print(el) # выведет 1, 2, 3 if el == 3: break

Инструкция break может завершать любые циклы: for, while.

Дано множество:

tst = {1, 3, 6, 7, -9, 12}

Выведите его элементы до первого отрицательного числа.

Дан список:

tst = [7, 1, 2, 5, 0, 3, 9]

Найдите сумму элементов этого списка до первого нуля.

Дано число:

tst = 897654

Сформируйте из него список до числа 6.
