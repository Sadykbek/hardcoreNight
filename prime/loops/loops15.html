Цикл while без счетчика в Python

В цикле while можно также выполнять какие-либо операции без счетчика, просто до тех пор, пока они будут истинны.

Давайте будем делить число 10.5 на 2, пока результат не станет меньше единицы. Для этого нужно задать обратное условие - цикл будет работать только, если число больше единицы:

num = 10.5 while num > 1: num = num / 2 print(num) # выведет 0.65625

С помощью цикла while делите число 100 на 3 столько раз, пока результат не будет меньше 20.

Пусть у вас есть число 1. С помощью цикла while умножайте его на 2.5 до тех пор, пока результат не станет больше 20.
