Слияние списка в строку в Python

Можно слить элементы списка в строку с заданным разделителем Это делается с помощью метода join. Метод применяется к строке, которая будет выступать в качестве разделителя, а в параметре метода указывается сливаемый список:

lst = ['1', '2', '3'] res = '/'.join(lst) print(res) # выведет '1/2/3'

Важный нюанс метода join состоит в том, что мы можем сливать только списки со строковыми элементами. В противном случае мы получим ошибку:

lst = [1, 2, 3] res = '/'.join(lst) print(res) # выведет ошибку

lst = ['a', 'b', 'c', 'd', 'e']
