Сортировка элементов в исходном списке в Python

Для сортировки элементов в списке применяется метод sort. Если оставить его параметр пустым, то элементы выстроятся по возрастанию. Если передать значение reverse=True, то элементы отсортируются по убыванию. Следует помнить, что после использования метода sort, меняется исходный список.

Давайте отсортируем элементы в списке по возрастанию:

lst = [3, 2, 1] lst.sort() print(lst) # выведет [1, 2, 3]

А теперь выстроим элементы по убыванию:

lst = [1, 2, 3] lst.sort(reverse=True) print(lst) # выведет [3, 2, 1]
