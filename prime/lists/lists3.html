Создание списков с помощью функции list в Python

Сделать список можно также с помощью функции list. В этом случае запись имеет чуть более длинную форму, чем при использовании квадратных скобок:

lst = list()

Далее в параметр функции list передаем элемент, который мы хотим добавить в список. Однако здесь есть важный нюанс. По своей сути функция list преобразует к списку элемент, указанный в ее параметре. Давайте сделаем список, состоящий из одной строки:

lst = list('1') print(lst) # выведет ['1']

Если мы хотим добавить в список несколько элементов, то передать их можем в виде одной длинной строки:

lst = list('1234') print(lst) # выведет ['1', '2', '3', '4']

При попытке передать в функцию число мы получим ошибку:

lst = list(1234) print(lst) # выведет ошибку

Дан следующий код:

tst = list('abcde') print(tst)
