Изменение элементов списка в Python

Элементы списка можно изменять также, как и символы строки:

lst = [1, 2, 3] lst[0] = '!' print(lst) # выведет ['!', 2, 3]

Дан список:

lst = [1, 2, 3, 4, 5]

Замените его второй элемент числом 10.

Дан список:

lst = ['a', 'b', 'c', 'd']

Замените последний элемент списка булевым значением.

Даны списки:

lst1 = [1, 2, 3] lst2 = ['a', 'b', 'c']

Замените последний элемент списка lst1 последним элементом lst2. Выведите первый список в консоль.
