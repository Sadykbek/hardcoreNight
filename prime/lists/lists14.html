Удаление элементов из списка оператором del в Python

С помощью оператора del можно удалить элемент из списка по его индексу:

lst = [1, 2, 3] del lst[0] print(lst) # выведет [2, 3]

После применения оператора del, все элементы, идущие после удаленного, сдвигаются вперед, чтобы занять его место. При этом длина самого списка становится меньше на 1.

Дан список:

lst = ['a', 'b', 'c', 'd', 'e']

Удалите элемент с номером 1.

Дан список:

lst = ['a', 1, 'b', 2, 'c', 3]

Оставьте в списке только строковые элементы.
