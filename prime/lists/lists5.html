Получение отдельного элемента списка в Python

Обратиться к элементам списка можно также как и к символам строки: первый элемент имеет номер 0, второй - номер 1 и так далее. Эти номера называются индексами элементов списка. Давайте посмотрим на примере:

lst = ['a', 'b', 'c', 'd', 'e'] print(lst[0]) # выведет 'a' print(lst[1]) # выведет 'b' print(lst[2]) # выведет 'c'

Дан список:

lst = [1, 2, 3, 4, 5]

Выведите первый элемент этого списка.

Дан список:

lst = [1, 2, 3, 4, 5]

Выведите второй элемент этого списка.

Дан следующий код:

lst = [1, 2, 3, 4, 5] print(lst[-1])

Скажите, что выведется в консоль.

Дан следующий код:

lst = [2, 4, 6, 8] res = lst[3] - lst[0] print(res)

Скажите, что выведется в консоль.
