Добавление элементов по позиции в список в Python

Мы также можем добавлять элементы в определенную позицию списка. Для этого используется метод insert. В его первый параметр передаем нужный номер позиции (индекс). Во втором параметре указываем добавляемый элемент:

lst = ['a', 'b', 'c', 'd'] lst.insert(1, '2') print(lst) # выведет ['a', '2', 'b', 'c', 'd']

Дан список:

lst = [1, 2, 3, 4, 5]

Добавьте после двойки элемент со значением 'a'.

Дан список:

lst = [1, 2, 3, 4, 5]

Добавьте в начало списка элемент со значением 'a'.

Дан список:

lst = [3, 6, 12]

Добавьте 9 перед числом 12 в списке.
