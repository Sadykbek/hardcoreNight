Добавление в список в Python

Мы также можем объединить два списка, изменив при этом исходный. В таких случаях мы применяем оператор +=:

lst1 = [1, 2, 3] lst2 = [4, 5, 6] lst1 += lst2 print(lst1) # выведет [1, 2, 3, 4, 5, 6]

Даны два списка:

lst1 = ['12', '34', '56'] lst2 = ['ab', 'cd', 'ef']

Объедините их.

Даны два списка:

lst1 = ['a', 'b', 'c'] lst2 = [4, 5, 6]

Добавьте ко второму списку первый.
