Объединение списков в Python

Для объединения списков можно применить оператор +. При этом мы получим новый список:

lst1 = [1, 2, 3] lst2 = [4, 5, 6] res = lst1 + lst2 print(res) # выведет [1, 2, 3, 4, 5, 6]

Даны списки:

lst1 = ['4', '5', '6'] lst2 = ['1', '2', '3']

Напишите код, чтобы получить следующий результат:

['1', '2', '3', '4', '5', '6']

Даны списки:

lst1 = ['a', 'b', 'c'] lst2 = ['d', 'e'] lst3 = [1, 2, 3]

Объедините их в новый список.
