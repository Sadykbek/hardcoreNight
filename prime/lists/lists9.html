Добавление элементов в конец списка в Python

Чтобы добавить элементы в конец списка, можно применить метод append. В параметр метода передаем нужный нам элемент:

lst = ['a', 'b', 'c'] lst.append('d') print(lst) # выведет ['a', 'b', 'c', 'd']

После применения метода мы увидим, что наш исходный список изменился.

Дан список:

lst = [1, 2, 3, 4, 5]

Добавьте ему в конец еще один элемент.

Дан список:

lst = []

Добавьте в него число 1.

Дан список:

lst = []

Последовательно заполните его строками: 'a', 'b', 'c'.
