Весь срез в Python

Чтобы получить полностью весь срез, можно применить оператор [:]:

txt = 'abcde' res = txt[:] print(res) # выведет 'abcde'

Такие срезы могут понадобиться для копирования исходного объекта. Давай скопируем нашу строку с помощью среза:

txt1 = 'abcde' txt2 = txt1[:] print(txt2) # выведет 'abcde'

Дан список:

lst = [1, 2, 3]

Скопируйте его.

Дано число:

num = 4567

Напишите код, чтобы получить следующий результат:

'4567'
