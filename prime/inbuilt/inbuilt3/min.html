Функция min

Функция min возвращает минимальное число из группы чисел, заданных в параметрах. Можно задать группу чисел, перечислив их через запятую, либо последовательность: список, кортеж, сет.

Синтаксис

min(группа чисел или последовательность)

Пример

Давайте выведем минимальное число из группы чисел:

num1 = 2 num2 = 5 num3 = 10 print(min(num1, num2, num3))

Результат выполнения кода:

2

Пример

А теперь давайте найдем минимальное число из группы отрицательных чисел:

num1 = -1 num2 = -3 num3 = -6 print(min(num1, num2, num3))

Результат выполнения кода:

-6

Пример

Также функция работает со списками. Давайте найдем минимальное число, передав список в параметр функции min:

lst = [10, 5, 4] print(min(lst))

Результат выполнения кода:

4

Пример

Давайте с помощью функции найдем минимальное число в кортеже:

tlp = (-2, 6, 9) print(min(tlp))

Результат выполнения кода:

-2
