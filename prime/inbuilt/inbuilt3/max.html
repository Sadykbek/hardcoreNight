Функция max

Функция max возвращает максимальное число из группы чисел, заданных в параметрах. Можно задать группу чисел, перечислив их через запятую, либо последовательность: список, кортеж, сет.

Синтаксис

max(группа чисел или последовательность)

Пример

Давайте выведем максимальное число из группы чисел:

num1 = 2 num2 = 5 num3 = 10 print(max(num1, num2, num3))

Результат выполнения кода:

10

Пример

А теперь давайте найдем максимальное число из группы отрицательных чисел:

num1 = -1 num2 = -3 num3 = -6 print(max(num1, num2, num3))

Результат выполнения кода:

-1

Пример

Также функция работает со списками. Давайте найдем максимальное число, передав список в параметр функции max:

lst = [10, 5, 4] print(max(lst))

Результат выполнения кода:

10

Пример

Давайте с помощью функции найдем максимальное число в кортеже:

tlp = (-2, 6, 9) print(max(tlp))

Результат выполнения кода:

9
