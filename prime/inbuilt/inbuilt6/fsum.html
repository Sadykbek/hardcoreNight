Метод fsum модуля math

Метод fsum модуля math возвращает сумму всех членов последовательности чисел. Тип возвращаемого значения всегда дробный.

Синтаксис

import math math.fsum(последовательность чисел)

Пример

Давайте найдем сумму всех членов последовательности чисел в списке:

import math lst = [1, 2, 3] print(math.fsum(lst))

Результат выполнения кода:

6.0

Пример

А теперь применим метод fsum к последовательности отрицательных чисел:

import math lst = [-2, -4, -6] print(math.fsum(lst))

Результат выполнения кода:

-12.0
