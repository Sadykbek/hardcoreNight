Метод sqrt модуля math

Метод sqrt модуля math возвращает квадратный корень указанного в параметре числа. Всегда возвращается вещественное значение.

Синтаксис

import math math.sqrt(число)

Пример

Давайте узнаем квадратный корень из числа 25:

import math print(math.sqrt(25))

Результат выполнения кода:

5.0

Пример

А теперь давайте получим квадратный корень из 10:

import math print(math.sqrt(10))

Результат выполнения кода:

3.1622776601683795

Пример

Давайте попробуем получить квадратный корень из отрицательного числа:

import math print(math.sqrt(-100))
