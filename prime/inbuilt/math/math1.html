Метод pow модуля math

Метод pow модуля math возвращает число, возведенное в степень. В первом параметре метода указываем нужное нам число, во втором параметре - в какую степень мы хотим его возвести. В отличие от оператора **, метод всегда возвращает вещественное число.

Синтаксис

import math math.pow(число, степень)

Пример

Давайте возведем число 3 в квадрат:

import math print(math.pow(3, 2))

Результат выполнения кода:

9.0

Пример

А теперь давайте возведем число в отрицательную степень:

import math print(math.pow(3, -2))

Результат выполнения кода:

0.1111111111111111

Пример

Давайте с помощью метода pow возведем в куб отрицательное число:

import math print(math.pow(-3, 3))

Результат выполнения кода:

-27.0
