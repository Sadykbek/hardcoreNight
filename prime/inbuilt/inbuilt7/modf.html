Метод modf модуля math

Метод modf модуля math возвращает кортеж, состоящий из дробной и целой частей заданного числа.

Синтаксис

import math math.modf(число)

Пример

Давайте узнаем дробную и целую часть числа 20:

import math print(math.modf(20))

Результат выполнения кода:

(0.0, 20.0)

Пример

А теперь применим метод modf к числу 20.56:

import math print(math.modf(20.56))

Результат выполнения кода:

(0.5599999999999987, 20.0)
