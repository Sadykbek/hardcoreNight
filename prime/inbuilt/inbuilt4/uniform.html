Метод uniform модуля random

Метод uniform модуля random возвращает псевдослучайное вещественное число из заданного диапазона. В первом параметре метода указываем начальное значение диапазона, во втором - конечное значение.

Синтаксис

import random random.uniform(начало, конец)

Пример

Давайте сгенерируем число в диапазоне от 0 до 10:

print(random.uniform(0, 10))

Результат выполнения кода:

4.840831811035069

Пример

А теперь давайте сгенерируем число в диапазоне отрицательных чисел:

print(random.uniform(-4, -1))

Результат выполнения кода:

-3.2516379041520467
