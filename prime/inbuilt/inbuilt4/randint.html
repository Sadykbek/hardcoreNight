Метод randint модуля random

Метод randint модуля random возвращает псевдослучайное целое число из заданного диапазона. В первом параметре метода указываем начальное значение диапазона, во втором - его конечное значение.

Синтаксис

import random random.randint(начало, конец)

Пример

Давайте сгенерируем число в диапазоне от 0 до 20:

print(random.randint(0, 20))

Результат выполнения кода:

19

Пример

А теперь давайте сгенерируем число в диапазоне отрицательных чисел:

print(random.randint(-15, -5))

Результат выполнения кода:

-6
