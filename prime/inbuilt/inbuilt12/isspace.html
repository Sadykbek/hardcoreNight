Метод isspace

Метод isspace проверяет входят ли в строку только пробелы. Помимо пробелов, метод распознает следующие символы - '\t' (горизонтальный отступ), '\n' (новая строка), '\v' (вертикальный отступ), '\f' (переход вниз к следующей странице), '\r' (возврат к началу строки). В параметры метода ничего не передается. Метод возвращает булевые значения True или False.

Синтаксис

строка.isspace()

Пример

Давайте проверим, что наша строка состоит только из пробельных символов:

txt = ' ' print(txt.isspace())

Результат выполнения кода:

True

Пример

Пусть теперь в строке есть иные символы:

txt = 'abc de' print(txt.isspace())

В результате мы увидим ошибку:

False

Пример

Давайте проверим вхождение в строку символа '\n':

txt = '\n' print(txt.isspace())

Результат выполнения кода:

True
