Объединение словарей с одинаковыми элементами в Python

Если в словарях находятся одинаковые элементы, то при объединении останется только один из них. Давайте посмотрим на примере. Пусть у нас есть следующие словари:

dct1 = { 'a': 1, 'b': 2, 'c': 3 } dct2 = { 'c': 3, 'e': 4, 'f': 5 }

Давайте объединим их:

dct1.update(dct2)

Посмотрим на результат:

print(dct1) # выведет {'a': 1, 'b': 2, 'c': 3, 'e': 4, 'f': 5}

Дан следующий код:

dct1 = { 'z': 2, 'w': 8, 'f': 10 } dct2 = { 'x': 4, 'y': 5, 'z': 6 } dct1.update(dct2) print(dct1)

Скажите, что выведется в консоль.

Дан следующий код:

dct1 = { 2: 'a', 4: 'b', 6: 'c' } dct2 = { 1: 'd', 3: 'e', 4: 'f', 7: 'j' } dct2.update(dct1) print(dct2)

Скажите, что выведется в консоль.
