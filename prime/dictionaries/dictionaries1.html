Словари в Python

Словарь представляет собой специальный тип хранения данных в виде пар ключ-значение.

Для создания словарей нужно использовать фигурные скобки. Давайте сделаем пустой словарь:

dct = {}

Теперь зададим строковые ключи и числовые значения:

dct = { 'a': 1, 'b': 2, 'c': 3 }

Дан словарь:

dct = {}

Добавьте в него три новых элемента.

Пусть у вас есть числа 2, 4 и 6. Создайте словарь, в котором эти числа станут ключами. При этом значениями должны быть строки 'ab', 'cd', 'ef'.

Создайте словарь с ключами surname, name, age. В значения словаря запишите ваши фамилию, имя и возраст.

Создайте словарь с месяцами. При этом пусть ключами будут порядковые номера месяцев, а значениями - их названия.
