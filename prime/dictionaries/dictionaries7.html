Объединение словарей в Python

Python предоставляет возможность объединить два словаря. Однако, если использовать оператор +, нам вернется ошибка.

Пусть у нас есть следующие словари:

dct1 = { 'a': 1, 'b': 2, 'c': 3 } dct2 = { 'e': 4, 'f': 5 }

Давайте попробуем их объединить с помощью оператора +:

print(dct1 + dct2) # выведет ошибку

Правильным решением будет применение метода update. В его параметре указываем словарь, которым будем дополнять исходный:

dct1.update(dct2) print(dct1) # выведет {'a': 1, 'b': 2, 'c': 3, 'e': 4, 'f': 5}

dct1 = { 'a': 1, 'b': 2, 'c': 3 } dct2 = { 'x': 4, 'y': 5, 'z': 6 }
