Получение всех ключей из словаря в Python

Чтобы получить все ключи из словаря, следует применить метод keys. В его параметре ничего не указывается. Возвращает метод специальный объект dict_keys.

Пусть у нас есть следующий словарь:

dct = { 'a': 1, 'b': 2, 'c': 3 }

Давайте выведем все ключи из него:

res = dct.keys() print(res) # выведет dict_keys(['a', 'b', 'c'])

Для удобства работы с объектом dict_keys можно преобразовать его в список. Это делается с помощью функции list:

res = list(dct.keys()) print(res) # выведет ['a', 'b', 'c']
