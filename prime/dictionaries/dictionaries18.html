Получение пары ключ-значение из словаря в Python

Из словаря можно получить также все элементы в виде списка кортежей, содержащих пары ключ-значение. Для этого применяется метод items, который возвращает специальный объект dict_items.

Давайте посмотрим на практике. Пусть у нас есть следующий словарь:

dct = { 'a': 1, 'b': 2, 'c': 3 }

Давайте выведем все элементы из него:

res = dct.items() print(res) # выведет dict_items([('a', 1), ('b', 2), ('c', 3)])

Объект dict_items можно преобразовать в настоящий список кортежей с помощью функции list:

res = list(dct.items()) print(res) # выведет [('a', 1), ('b', 2), ('c', 3)]
