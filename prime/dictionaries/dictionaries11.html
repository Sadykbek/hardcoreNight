Извлечение последнего элемента в Python

С помощью метода popitem можно извлечь последний элемент из словаря. Он возвращает кортеж из пары ключ-значение извлеченного элемента.

Давайте попробуем на практике. Пусть у нас есть следующий словарь:

dct = { 'a': 1, 'b': 2, 'c': 3 }

Давайте извлечем последний элемент из словаря:

print(dct.popitem()) # выведет ('c', 3)

А из словаря этот элемент исчезнет:

print(dct) # выведет {'a': 1, 'b': 2}

Дан словарь:

dct = { 'x': 1, 'y': 2, 'z': 3 }

Получите из него кортеж из последнего элемента с ключом, удалив его из словаря.

Дан словарь:

dct = { 1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e' }

Напишите код, чтобы получить следующий результат:

[4, 3, 2, 1]
