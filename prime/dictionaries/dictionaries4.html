Изменение значения элемента словаря в Python

Изменить значение элемента словаря можно по его ключу. Для этого после указания ключа после оператора = прописываем новое значение.

Давайте посмотрим на примере. Пусть у нас есть следующий словарь:

dct = { 'a': 1, 'b': 2, 'c': 3 }

Давайте изменим его элемент:

dct['a'] = '!'

Посмотрим на изменившийся словарь:

print(dct) # выведет {'a': '!', 'b': 2, 'c': 3}

Дан словарь:

dct = { 'x': 1, 'y': 2, 'z': 3 }

Увеличьте элемент с ключом 'x' на единицу.

Дан словарь:

dct = { 'surn': 'smit', 'name': 'john' }

Измените значения для ключей 'surn' и 'name' на свои фамилию и имя.
