Отдельный элемент кортежа в Python

Чтобы получить элемент кортежа, нужно обратиться к нему по индексу:

tpl = ('a', 'b', 'c') print(tpl[0]) # выведет 'a'

Дан кортеж:

tpl = (1, 2, 3, 4)

Выведите из него первый элемент.

Дан кортеж:

tpl = ('a', 'b', 3, 4, 'c')

Выведите из него третий элемент.

Дан кортеж:

tpl = (10, 9, 8, 7, 6)

Выведите из него последний элемент.

Из кортежа, заданного в предыдущей задачи, выведите предпоследний элемент.

Дан кортеж:

tpl = (1, 2, 3)

Найдите сумму элементов этого кортежа.
