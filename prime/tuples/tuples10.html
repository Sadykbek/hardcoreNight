Умножение кортежей в Python

Кортежи также можем умножать, т.е. копировать их определенное количество раз. Делается с помощью оператора *:

tpl = ('a', 'b') res = tpl * 2 print(res) # выведет ('a', 'b', 'a', 'b')

Дан кортеж:

tpl1 = ('1', '2', '3')

Создайте из него следующий кортеж:

tpl2 = ('1', '2', '3', '1', '2', '3', '1', '2', '3')

Даны кортежи:

tpl1 = ('a', 'b') tpl2 = (1, 2)

Напишите код, чтобы получить следующий кортеж:

tpl3 = ('a', 'b', 'a', 'b', 1, 2)
