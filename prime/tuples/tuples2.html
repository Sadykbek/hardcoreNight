Создание кортежа в Python

Кортежи можно создавать просто с помощью круглых скобок. Для этого достаточно объявить переменную и заключить ее значение в круглые скобки. Давайте создадим пустой кортеж:

tpl = ()

Теперь можно перечислить элементы кортежа внутри круглых скобок:

tpl = ('a', 'b', 'c')

Кортежи могут включать элементы разных типов: строки, числа, булевые значения, списки:

tpl = ('1', '2', '3') # строки tpl = (1, 2, 3) # числа tpl = (True, False) # булевые значения tpl = (['a', 'b'], ['c', 'd']) # списки

Дана переменная:

tst = (12, 34, 45)

Скажите, к какому типу данных она относится.

Дана переменная:

tst = [1, 2, 3]

Скажите, к какому типу данных она относится.

Дана переменная:

tst = ([1, 2, 3], True, 'a', 'b')

Скажите, к какому типу данных она относится.
