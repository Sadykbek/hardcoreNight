Наличие элемента в кортеже в Python

Узнать есть ли данный элемент в списке можно с помощью оператора in:

tpl = ('a', 'b' 'c') res = 'a' in tpl print(res) # выведет True

Чтобы проверить отсутствие элемента, следует использовать операторы not in:

tpl = ('a', 'b' 'c') res = 'a' not in tpl print(res) # выведет False

Дан кортеж:

tpl = (2, 4, 6, 10)

Проверьте, есть ли в нем число 8.

Дан кортеж:

tpl = ('abc', 'def')

Проверьте, есть ли в нем строка 'd'.

Дан следующий код:

tpl = ('1', '2', '3') res = 1 not in tpl print(res)
