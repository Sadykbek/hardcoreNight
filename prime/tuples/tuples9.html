Объединение кортежей в Python

Для объединения кортежей можно применить оператор +:

tpl1 = ('a', 'b', 'c') tpl2 = ('d', 'e') res = tpl1 + tpl2 print(res) # выведет ('a', 'b', 'c', 'd', 'e')

Даны три кортежа:

tpl1 = ('1', '2', '3') tpl2 = ('4', '5', '6') tpl3 = ('7', '8', '9')

Слейте элементы этих кортежей в один новый кортеж.

Даны кортежи:

tpl1 = (3, 4) tpl2 = (1, 2)

Слейте их так, чтобы получить следующий кортеж:

tpl3 = (1, 2, 3, 4)
